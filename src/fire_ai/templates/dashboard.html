How To Use This

  • The system now reads simulated live sensor values automatically and predicts the fire state continuously.
  • You do not need to choose Normal, Warning, or Fire during normal use. The model decides that from the incoming readings.
  • `Reset Sensor History` clears the rolling memory so the next readings start from a fresh baseline.
  • `Pause Auto Feed` is only for demo control if you want to stop the live stream temporarily.
