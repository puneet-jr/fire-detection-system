How To Use This

  • Dataset mode reads rows from your dataset, runs the feature builder and model, and gives a short alert sound when flame or strong fire evidence is detected.
  • Live hardware mode listens to the Arduino Uno, shows the real sensor values, logs detections, and drives the physical buzzer when fire evidence is detected.
  • When the flame sensor becomes `1` during live hardware mode, the Arduino buzzer should sound immediately and the dashboard will log the event.
  • Only one mode runs at a time. Starting one mode stops the other mode first.
  • The chart below updates in real time for whichever mode is active, so you can see temperature, humidity, and flame-trigger points as they happen.
